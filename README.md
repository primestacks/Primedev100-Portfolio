My portflio
site built with Vite, React, and TailwindCSS.
Features a dark theme with custom colors and gradient effects.
Includes sections for About, Projects, Skills, and Contact information.

## Technologies Used

- Vite
- React
- TailwindCSS
- Oklch Color Scheme

## Setup Instructions

1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies using `npm install`.
4. Start the development server with `npm run dev`.
5. Open your browser and go to `http://localhost:3000` to view the site.
